# E-commerce Lifecycle Analytics

An end-to-end analytics project analyzing **customer behavior across the e-commerce lifecycle** using event-level interaction data.

The project explores how users interact with an online store, how they move through the **conversion funnel**, how frequently they return, and how different **customer segments behave**.

The goal is to identify **conversion bottlenecks, engagement patterns, and business opportunities** that can improve customer experience and revenue.

---

# Project Overview

This project analyzes **~1,999,840 user interaction events generated by ~935,717 users** on an e-commerce platform.

Each interaction represents a user event such as:

* viewing a product
* adding a product to cart
* completing a purchase

The analysis focuses on three core questions:

• Where do users drop off in the purchase funnel?
• How frequently do users return to the platform?
• What different types of customers exist on the platform?

---

# Key Insights

### Conversion Funnel

The customer journey follows the typical e-commerce funnel:

View → Cart → Purchase

The analysis shows that the **largest drop-off occurs between product views and cart additions**, suggesting that product pages may not effectively convert browsing users into buyers.

However, once users add products to their cart, the probability of completing a purchase increases significantly.

---

### User Engagement

User engagement metrics show moderate platform interaction:

* Average sessions per user: **1.68**
* Same-day repeat users: **29.97%**
* 7-day return rate: **16.28%**

This suggests that while users frequently explore the platform, long-term retention could be improved.

---

### Customer Segmentation

Behavioral segmentation identified multiple user groups:

• **Browsers** – users who primarily view products
• **Cart Users** – users demonstrating purchase intent
• **Purchasers** – users completing transactions
• **Low Engagement Users** – minimal activity users

Understanding these segments allows businesses to apply **targeted marketing strategies and personalized recommendations**.
